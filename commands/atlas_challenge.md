# /atlas:challenge

Run a Socratic challenge on a plan without executing it.

Activates the Critic and Devil's Advocate agents internally:
- Critic: finds flaws, missing edge cases, hidden assumptions
- Devil's Advocate: proposes alternatives, challenges the framing

Produces a Socratic Summary with a verdict:
- PROCEED
- PROCEED WITH CAUTION
- REDESIGN

Use this before committing to any significant architectural or implementation decision.
