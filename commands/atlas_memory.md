# /atlas:memory

Display what ATLAS currently knows about this project.

Reads `.atlas/memory.json` via the atlas-mcp MCP server and presents it in a clear, readable format:

- Project name, description, stack
- Architecture overview and key decisions
- Critical files, fragile areas, stable modules
- Known gotchas and open questions
- Last session summary

Use this to audit ATLAS's knowledge or share context at the start of a session.
