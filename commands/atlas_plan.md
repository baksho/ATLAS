# /atlas:plan

Run a full strategic planning session with Socratic challenge.

Steps:
1. Load memory context from `.atlas/memory.json`
2. Draft a step-by-step plan for the goal
3. Run Critic + Devil's Advocate on the draft
4. Revise the plan based on findings
5. Tag each step with confidence level
6. Present the final plan and wait for approval

Use this before starting any complex or multi-step task.
